# 🧠 Autism Detection using Machine Learning

## 📌 Overview
This project is a Machine Learning-based web application that predicts the likelihood of Autism Spectrum Disorder (ASD) based on behavioral and personal input features.

## 🚀 Live Demo
👉 https://autism-detection-using-ml-f75taqclkvbzrrcgue5uqd.streamlit.app/

## 🧩 Features
- Autism prediction using ML model
- Interactive web interface (Streamlit)
- Fast and real-time predictions
- Data preprocessing and model training
- Clean and user-friendly UI

## 🛠 Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy
- Streamlit
- Pickle
