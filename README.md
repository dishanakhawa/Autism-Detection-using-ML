# Autism Detection using Machine Learning

This project is a Machine Learning-based web application that predicts the likelihood of Autism Spectrum Disorder (ASD) based on user input features.

## Overview
The system takes behavioral and personal attributes as input and predicts whether a person is likely to show signs of autism using a trained ML model.

## Features
- Autism prediction using ML
- User-friendly interface
- Fast and simple predictions
- Data preprocessing and model training
- Clean and interactive UI

## Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy
- Streamlit / Flask
- Pickle
